<h2>IMP 2.0.0 stable release</h2>

<p>This release is recommended for beginners, and should be stable.
(For the very latest features, use the
<a href="#nightly">nightly builds</a> instead.)</p>

<p>All of the RPMs are signed with

For all other packages, please check the
<a href="http://salilab.org/imp/2.0.0/download/MD5SUM">MD5SUM</a> and
<a href="http://salilab.org/imp/2.0.0/download/MD5SUM.asc">its signature</a>.
</p>

<a name="nightly"></a>
<h2>Nightly builds</h2>

<p>Nightly builds are not as well tested as the
<a href="#stable">stable releases</a> but do include new functionality
(they are updated daily).</p>

<h3>Installers and source code</h3>

<p>Installers for Windows, Linux (Fedora 18, RHEL 5, RHEL 6), and Mac OS X 10.6
(Snow Leopard) <a href="http://salilab.org/imp/nightly/download/">are
available</a>. The source code can also be obtained from the same location,
or <a
href="http://salilab.org/imp/nightly/doc/html/installation.html#devbuild">from
our Subversion repository</a>.</p>

<h3>Documentation</h3>

<p>Please refer to the
<a href="http://salilab.org/imp/nightly/doc/html/">online documentation</a>.
</p>
